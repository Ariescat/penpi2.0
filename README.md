# penpi2.0

个人在校期间的参赛以及毕设作品，完成了大部分功能的技术开发。

2.0新特性：

  - 在原有的“校园快递最后一公里”的功能上，拓展出了社区功能。
  - 服务端有更完整的交互接口和后台管理功能，基于Spring，采用Gradle构建，并用了Elastic Search和Disruptor等技术。
  - 客户端采用weex技术全新构建了一次。
